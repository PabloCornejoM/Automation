# Automation Learning Repository

A collection of notebooks exploring fundamental automation tools and techniques in Python.

## Contents

### Notebooks

1. **PyAutoGUI** - Learning GUI automation and mouse/keyboard control
2. **Bash** - Working with shell commands and subprocess integration
3. **Regular Expressions** - Pattern matching and text processing
4. **Cron** - Scheduling tasks and automation

### Data

The `Expo_2021/` directory contains monthly datasets (CSV, DTA, SAV formats) from 2021.

## Setup

### Requirements

- Python 3.11
- Dependencies: numpy, pillow, opencv, pyautogui, keyboard, mouse, pygetwindow, pymsgbox, crontab

### Creating the Environment

```bash
conda env create -f environment.yml
conda activate automation_venv
```

## Usage

Open the notebooks in Jupyter to explore each topic with working examples.

## Files

- `test.py` - Utility script for cron testing
- `environment.yml` - Conda environment configuration
